Gra w Kółko-krzyżyk


Opis
Gra w Kółko-krzyżyk (XOX) to klasyczna gra planszowa dla dwóch graczy. Celem gry jest ułożenie trzech swoich znaków w linii (poziomo, pionowo lub diagonalnie) na planszy 3x3. Gracz 1 używa znaku 'O', a Gracz 2 znaku 'X'.

Funkcjonalności
Interaktywne menu z zasadami gry.
Wyświetlanie planszy po każdym ruchu.
Wprowadzanie ruchów przez graczy.
Sprawdzanie zwycięzcy lub remisu po każdym ruchu.

Jak uruchomić

Skopiuj kod do pliku XOX.cpp.
Skompiluj plik za pomocą kompilatora C++ (np. g++):
```g++ XOX.cpp -o XOX```
Uruchom program:
```./XOX```

Wymagania

Kompilator C++ (np. g++, clang++),
podstawowa znajomość obsługi terminala
